1. При запуске сервера и клиента на одном компьютере вместо ip напишите "localhost" нажмите "connect".
2. Введите запрос в поле напротив кнопки "Enter" и нажмите "Enter":)
3. Для просмотра текста файла нажмите на файл в списке ответов :)

- Также можно отправить запрос ввиде json файла, пример такого файла есть в папке "config" - "Requests.json"
